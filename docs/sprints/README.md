# 🏃‍♂️ Sprint Tracking

Thư mục này dùng để lưu trữ và theo dõi các công việc (tasks) theo từng Sprint của dự án. 

## 📂 Cấu trúc thư mục
* `sprint-template.md`: Template mẫu để copy khi tạo Sprint mới.
* `sprint-01.md`, `sprint-02.md`, ...: File chi tiết của từng Sprint.

## 🔄 Quy trình làm việc (Workflow)
1. **Trước Sprint Planning**: Copy file `sprint-template.md` thành file mới (ví dụ: `sprint-01.md`).
2. **Trong Sprint Planning**: Cùng team chốt **Sprint Goal**, liệt kê các task cần làm vào bảng công việc và gán người thực hiện (Assignee).
3. **Trong Sprint**: Cập nhật trạng thái (`Todo` -> `In Progress` -> `In Review` -> `Done`) và gắn link Pull Request tương ứng vào bảng.
4. **Cuối Sprint (Retrospective)**: Điền các thông tin tổng kết, bài học rút ra và các hành động cải tiến vào cuối file.
